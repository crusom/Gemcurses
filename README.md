# Gemcurses
A Gemini client, written in C using ncurses

| KEY| ACTION        |
| -- | ------------- |
| arrows up/down | go down or up on the page      |
| PgUp/PgDn      | go page up or page down        |
| q              | change to link-mode/scroll-mode|
| /              | search                         |
| enter          | go to a link                   |  
| B              | go to the defined main gemsite |
| mouse scroll   | scroll                         |
